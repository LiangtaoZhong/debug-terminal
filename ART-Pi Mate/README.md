# ART-Pi Mate

## 包含功能

1. SPI LCD
2. UART*2
3. CAN*1
4. IIC*2
5. KEY*2
6. LED*2
7. IO扩展接口*6
8. 蜂鸣器*1
9. 无线透传
10. NRF24L01
